## Base URL

`https://api.verisecid.com`

All endpoints are under `/api/v1/kyc/...`.

## Authentication

Use your public API key in the Authorization header:

```http
Authorization: Bearer pb-YourPublicKey
```

Public keys map to users in Firestore at `users.apiPublicKey`.

## Endpoints

- Create hosted session — `POST /api/v1/kyc/sessions`
- Get session — `GET /api/v1/kyc/sessions/{sessionId}`
