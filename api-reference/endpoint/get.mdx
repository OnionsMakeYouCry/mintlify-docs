title: 'Get KYC Session'
description: 'Retrieve a single hosted session by ID'
---

## Endpoint

`GET /api/v1/kyc/sessions/{sessionId}`

## Authentication

```http
Authorization: Bearer pb-YourPublicKey
```

## cURL

```bash
curl -X GET "https://api.verisecid.com/api/v1/kyc/sessions/SESSION_ID" \
  -H "Authorization: Bearer pb-YourPublicKey"
```

## Success response (200)

```json
{
  "sessionId": "<session-id>",
  "workspaceId": "<workspace-id>",
  "userId": "<uid>",
  "status": "not_started",
  "redirectUrl": null,
  "metadata": { "customerId": "12345" },
  "createdAt": "2025-08-09T10:00:00.000Z",
  "updatedAt": "2025-08-09T10:05:00.000Z",
  "verificationUrl": "https://<host>/verify/<session-id>"
}
```

## Authorization rules

- Requesting user must be the `userId` owner OR owner of `workspaceId`

## Errors

- 401 missing/invalid auth
- 403 forbidden
- 404 not_found
